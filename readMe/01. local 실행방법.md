## 02. H2 실행 및 연결
1.H2 실행

2.jdbc url : jdbc:h2:~/tourDB 로 변경 -> '연결' 클릭

3.좌상단) '연결끊기' 클릭

4.jdbc url : jdbc:h2:tcp://localhost/~/tourDB 로 변경 -> '연결' 클릭
